Another Dungeon Explorer (NEA)

Description
An educational Python game project built with Pygame, designed to teach advanced game mechanics: smooth camera movement, boss AI, enemy collisions, and dynamic UI. Players explore dungeons, fight enemies, collect items, and face challenging bosses.

Features

Smooth and zoomable camera system
Player combat with attack hitboxes
Enemy AI: zombies, rats, and bosses with unique attack patterns
Health and item pickups (potions, hazards)
Tile-based levels using TMX maps
Animated sprites for characters and objects
Educational code: modular classes for Level, Player, Enemies, Items, and Timers

Controls:

Arrow Keys – Move

Space – Jump

X – Attack

Down – Drop through platforms

Installation

Clone the repository:
git clone https://github.com/rjd14me/NEA.git
Install dependencies:
pip install pygame pytmx
Run the game:
python main.py

Project Structure

NEA/

├── Levels/          # TMX maps for levels
