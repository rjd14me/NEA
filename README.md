# Another Dungeon Explorer Game (NEA)

## Description:
A simple python 2d side scroller dungeon Explorer game inspired by works such as Hollow Knight


## Controls:
- **Arrow Keys** – Move
- **Space** – Jump
- **X** – Attack
- **Down** – Drop through platforms


## Installation:

## Clone the repository:
```bash
git clone https://github.com/rjd14me/NEA.git
Install dependencies:
pip install pygame pytmx
Run the game:
python AnotherDungeonExplorerGamev.py
```

## Project Structure:
```bash
NEA/
├── Assets/          # Sprite sheets, props, enemies, items
├── Levels/          # TMX maps for levels
├── main.py          # Game entry point
├── README.md
```
